```

├─────────────────────────────────────────┤
```

## 技術仕様

- **ウィンドウサイズ**: 400x300ピクセル（初期値）
- **最小サイズ**: 300x200ピクセル
- **レイアウト**: StackPane（中央配置）
- **リサイズ**: 可能
- **フォント**: システムデフォルト、24px、太字

## ファイル構成

- `BasicWindowsApp.java`: メインアプリケーションクラス
- `pom.xml`: Maven設定ファイル
- `README.md`: 使用説明書
